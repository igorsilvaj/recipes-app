## About The Project
Projeto em grupo realizado durante o curso de FrontEnd da Trybe.

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:igorsilvaj/recipes-app.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the application
   ```js
   npm start
   ```
